## Overview

CrossOver Advanced is a Windows application compatibility tool from CodeWeavers that runs Windows software directly on macOS without requiring a Windows license, virtual machine, or reboot, using a sophisticated compatibility layer based on Wine technology that translates Windows API calls to native macOS equivalents in real time. It eliminates the overhead of maintaining a full Windows installation — no Windows license cost, no virtual machine memory allocation, no disk partition dedicated to a second operating system. Windows applications run as native processes with direct hardware access, delivering performance that virtual machines cannot match for compatible software.

The compatibility layer translates Windows system calls, DirectX graphics commands, and Windows API requests to their macOS equivalents at runtime, enabling Windows executables to run without modification on hardware and operating systems they were never designed for. CrossTie technology automates the installation of supported applications with one-click setup that handles all configuration, dependencies, and compatibility settings automatically. The translation layer is continuously updated to support new applications and improve existing compatibility.

Bottle management creates isolated Windows-like environments for each application or group of related applications, preventing conflicts between applications that require different Windows configurations or library versions. Each bottle operates as an independent Windows environment with its own registry, file system, and configuration without affecting other bottles or the host system. Bottles can be backed up, exported, and shared, making it practical to distribute pre-configured application environments across teams.

Application compatibility covers a broad range of Windows productivity software, games, and specialized tools with a searchable compatibility database that reports the current status and any configuration requirements for thousands of tested applications. Community ratings supplement official testing with real-user experience reports that cover edge cases and specific use scenarios. CodeWeavers actively works with high-profile applications to improve compatibility, prioritizing the software that users most need.

For users who need specific Windows applications but do not want the cost, complexity, and resource overhead of maintaining a full Windows installation through virtualization, CrossOver Advanced is the lightweight alternative that runs Windows software as native processes without a virtual machine. Its Wine-based approach means zero Windows license fees, zero boot time, and performance that benefits from direct hardware access.

## Performance

CrossOver balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
